Once you have the boxes being added correctly, you will need to do the following:

  1. Make sure the user can not enter a value for the x or y coordinates that is valid. Valid means the box must fit completely insode the box container.
  2. Add a SELECT box (drop down list) with at least 4 colors. When the user clicks "Add Box", use the selected color as the background color.
  3. Change the select element to be a "color" input. This will allow the user to choose any color they want.
  4. When the user clicks on each "box" it will remove it from the DOM
  5. When the user hovers over the box, move it to a random location in the container.
  6. If the user enters no values, then add a box with a random color and x/y coordinates
  7. Add a random z-index value between 1 and 20 to each box.

Extra Credit

Have each box randomly move and change background colors every 5 seconds.
